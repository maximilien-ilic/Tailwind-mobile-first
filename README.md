# Tailwind-mobile-first

Ce projet est une reproduction fidèle d’une maquette en utilisant Vue.js et Tailwind CSS. L’approche est mobile-first, ce qui garantit une expérience optimale sur smartphone avant d’adapter le design aux écrans plus larges.

L’objectif principal était de pratiquer l’intégration de maquettes, de structurer des composants Vue et d’utiliser efficacement Tailwind CSS pour le styling réactif.

🛠️ Technologies utilisées

Vue.js
 – Framework JavaScript pour construire l’interface utilisateur.

Tailwind CSS
 – Framework CSS utility-first pour un design rapide et responsive.

Nuxt – pour le bundling et le développement rapide.

📱 Fonctionnalités

Reproduction complète de la maquette fournie.

Design mobile-first avec adaptations pour tablettes et desktop.

Composants Vue modulaires et réutilisables.

Navigation et layout responsive.
